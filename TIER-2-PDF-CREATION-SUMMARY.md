## How to Create the PDF

### Option 1: Browser Print (Recommended)
1. Open `TIER-2-VENDOR-REGISTRATION-DIRECTORY-PDF-READY.html` in your web browser
2. Press **Cmd+P** (Mac) or **Ctrl+P** (Windows)
3. In the print dialog:
   - Select **"Save as PDF"** or **"Microsoft Print to PDF"**
   - Set margins to **"Default"** or **"Minimum"**
   - Ensure **"Background graphics"** is enabled
4. Click **"Save"** and choose your location
5. Suggested filename: `Tier-2-Vendor-Registration-Directory.pdf`

### Option 2: Using Chrome DevTools
1. Open the HTML file in Google Chrome
2. Press **F12** to open DevTools
3. Press **Cmd+Shift+P** (Mac) or **Ctrl+Shift+P** (Windows)
4. Type "PDF" and select **"Print to PDF"**
5. Choose save location
